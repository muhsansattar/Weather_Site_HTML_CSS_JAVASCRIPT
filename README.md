## My Web Project

## Weather Site

Welcome to the Weather Site project! This web application provides users with up-to-date weather information for various locations. Built with HTML, CSS, and JavaScript, the Weather Site offers a user-friendly interface and real-time weather data to enhance the user experience.

[Video.webm](https://github.com/user-attachments/assets/b39e0553-a2ea-41ce-8998-fdf98a6d5dc5)

## Features
- Real-Time Weather Updates: Get current weather information based on user input.
- User-Friendly Interface: Intuitive design for easy navigation and accessibility.
- Responsive Design: Optimized for various devices, ensuring a seamless experience on desktops, tablets, and smartphones.
- Interactive Elements: Engaging features that allow users to search for and view weather data.

![Full-Screen-Capture](https://github.com/user-attachments/assets/88b3831d-5a93-4177-b0a2-c1d6086560d3)

## Technology Stack
- HTML: For structuring the content of the website.
- CSS: For styling and ensuring a visually appealing layout.
- JavaScript: To handle API calls and manage dynamic updates for weather data.
